# Java 入门 -- 基本程序设计

java程序基于类。一个基础的java程序如下:
```java
public class HelloWorld{
  public static void main(String[] args){
    System.out.println("Hello World");
  }
}
```

<!--more-->
java程序的入口是main函数。约定main函数为static 的，参数为``` String[] ```。
